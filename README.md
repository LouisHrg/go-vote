## Requirements :

go 1.13

## Run the project

Install go vendor :

```make install-go-vendor```

Install all the dependencies :

```make install-dep```

Build and run the project :

```make run```

Simply start a builded version of the project :

```make start```

You can run static analysis of the base code with :

```make analysis```
